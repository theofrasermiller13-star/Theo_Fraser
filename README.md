# Landing Page Template

This repository is designed to teach students the foundational technologies of the web from the ground up.

## Overview

The project provides a structured learning environment for students to explore core web development concepts, build practical skills, and progress through the fundamentals in a clear and organised way.

## Getting Started

To run the project locally, use:

```bash
python3 -m http.server 3000
```

Then open your browser and navigate to:

```text
http://localhost:3000
```

## Purpose

The goal of this repository is to support students as they learn essential web technologies step by step, with a focus on building a strong foundation.

## Features

- Beginner-friendly structure.
- Clear progression through foundational web concepts.
- Simple local development setup.

## Notes

Make sure you have Python 3 installed before running the server command(GitHub Codespaces usually comes with Python 3 preinstalled).
